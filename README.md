# Advent of Code 2025

Rust solutions for Advent of Code 2025.

## Usage

### Run all days
```bash
cargo run --release
```

### Run a specific day
```bash
cargo run --release -- 1
```

### Run with test input
```bash
# All days with test input
cargo run --release -- --test

# Specific day with test input
cargo run --release -- 1 --test
```
